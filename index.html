About Alpha Download

Design is easy to start and painful to finish.

Fovey exists to fix the finish.

Fovey is a modern vector design tool built for designers who care about what happens after the artboard. Not just how things look, but how work is reviewed, handed off, and actually used in the real world.

We are building Fovey for people who design with intent, not just aesthetics.

Who We Are

Fovey is an independent design platform built by designers and developers who have lived inside the cracks of existing tools.

We have shipped work to clients.
We have handed off files that came back broken.
We have watched beautiful designs fall apart at the moment of delivery.

Fovey is the product of that frustration.

This is not a clone of existing vector apps. It is a rethink of how design ownership, structure, and delivery should work from day one.

The Gap Fovey Fills

Most design tools stop caring the moment the pixels are placed.

They are excellent at creation and vague at everything else.

Fovey focuses on what other tools treat as an afterthought:

  • • Structured brand handoff, not scattered files
  • • Clear ownership of assets, not hidden dependencies
  • • Review and packaging built into the design process
  • • A path from canvas to client without chaos

Instead of asking designers to explain their work after exporting, Fovey helps designers ship work that explains itself.

This is where Fovey is different.
It is not just another canvas.
It is a delivery system for design intent.

What's Included in the Alpha

The alpha is focused, opinionated, and intentionally incomplete.

You will have access to:

  • • A fully functional vector canvas
  • • Shapes, paths, text, and direct selection tools
  • • Google Fonts integration with live previews
  • • Asset uploading and reuse
  • • Frames for organizing work and exports
  • • SVG, PNG, and JPEG export
  • • Brand-ready structure, not loose artboards
  • • A fast local file system with no account required

What you will not have yet:

  • • Collaboration
  • • Cloud sync
  • • Advanced effects and filters
  • • Accounts or publishing workflows

This alpha is about proving the core loop works. Creation, structure, export, and handoff.

What We Expect From Alpha Users

Alpha access is not early access for spectators.

We expect:

  • • Real usage, not just curiosity installs
  • • Honest feedback, including when something feels wrong
  • • Bug reports with context, not vibes
  • • Willingness to tolerate rough edges

You are helping shape the product. That means breaking it, questioning decisions, and pushing it into real workflows.

If you want polished marketing demos, wait for beta.
If you want influence, alpha is for you.

How to Install and Run Fovey

Windows Instructions

  1. 1. Download the zip file
  2. 2. Right-click the zip and select "Extract All"
  3. 3. Open the extracted folder
  4. 4. Double-click the Fovey application
  5. 5. If Windows SmartScreen appears, click "More info" then "Run anyway"

Fovey runs locally on your machine. No installer required.

macOS Instructions

  1. 1. Download the zip file
  2. 2. Double-click to extract it
  3. 3. Drag the Fovey app into your Applications folder
  4. 4. Right-click the app and select "Open"
  5. 5. Confirm you want to open an unsigned application

macOS blocks unsigned apps by default. This is expected during alpha.

Your Work Is Yours

Fovey does not claim ownership over your designs.

  • • All assets you create or import remain yours.
  • • Files are stored locally on your machine.
  • • There is no account and no upload happening behind your back.

When cloud features arrive, ownership will remain non-negotiable.

Where This Is Going
